# Card Demo Generated Assets

Generated with the built-in image generation tool for the MVP card encounter demo.

## Files

- `card_demo_icon_sheet.png`
  - 4x4 icon sheet for early card/event concepts.
  - Suggested cell meanings, row-major:
    weak healing, private sermon, dream whisper, old well,
    hidden tracks, ritual circle, plague miasma, death echo,
    grateful family, suspicious monk, fever nightmare, faith spark,
    bone shard, anchor sigil, exposed eye, sanity moon.
- `card_demo_card_frames.png`
  - 2x2 card frame concept sheet.
  - Suggested use:
    action card, ritual card, corruption/status card, card back.

## Notes

- These are concept/demo assets, not final sliced sprites.
- The icon sheet uses a checker-like dark background, not true transparency.
- If individual transparent icons are needed, split the sheet into cells and remove the background in a later pass.
